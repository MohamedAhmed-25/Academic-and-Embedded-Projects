# Academic, AI & Embedded Systems Projects 🚀

A curated collection of academic, hardware-software integration, and utility projects developed during my first year in Computer Science.

## 1. Fire Guard AI 🔥
* **Description:** An intelligent real-time fire and smoke detection system leveraging computer vision and deep learning.
* **Tech Stack:** Python, OpenCV, TensorFlow, Keras.
* **Key Features:** Real-time monitoring via camera feeds with quick response alert triggers.

## 2. Smart Security Lock System (Biometrics) 🔒
* **Description:** A multi-layered hardware security prototype integrating biometric authentication with manual passcodes.
* **Tech Stack:** Embedded Systems, C++, Arduino.
* **Key Features:** Integrates a password-protected keypad and a fingerprint sensor for responsive access control.

## 3. Smart Street Lighting System (Energy Efficiency) 💡
* **Description:** An intelligent street lighting prototype (Maquette) designed for maximum energy conservation in smart cities (Physics Project).
* **Tech Stack:** Embedded Systems, C++, Arduino.
* **Key Features:** Integrates LDR sensors for ambient light detection and PIR motion sensors to trigger high-intensity lighting only when movement is detected.

## 4. Student Grade Management System 🎓
* **Description:** A console application built using structured programming to handle basic academic administrative tasks.
* **Tech Stack:** C++.
* **Key Features:** Allows seamless adding, searching, calculating grades, and updating student records.
